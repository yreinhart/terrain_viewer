# Terrain Viewer

Онлайн-сервис для анализа участка по XLSX-файлу с координатами `X`, `Y`, `H`.

## Реализованные визуализации

### Рельеф
- Карта высот
- Карта высот с сеткой
- Горизонтали
- Карта высот + горизонтали
- Карта уклонов
- Карта направления уклона

### Подсыпка
- Карта толщины подсыпки
- Карта выемка / насыпь

### Вода
- Сток воды D8
- Подтопление по уровню
- Время стекания
- Потенциальные водотоки
- Анимация подтопления

### Профили
- Продольный профиль
- Поперечный профиль
- Разрез по двум точкам

### Контроль данных
- Карта качества данных
- Карта неопределенности интерполяции

### Планирование
- Карта допустимых зон
- Маршрут техники

## Важные ограничения

Это инструмент предварительной оценки по цифровой сетке высот. Модули воды, маршрута и пригодности не заменяют проектирование дренажа, геологию, инженерные изыскания или расчет устойчивости откосов.


## Формат XLSX

Файл должен содержать колонки:

| X | Y | H |
|---|---|---|
| 0 | 0 | -1.01 |
| 1 | 0 | -1.25 |

Где:

- `X` - координата по X, м;
- `Y` - координата по Y, м;
- `H` - отметка высоты, м относительно нуля проекта.

## Локальный запуск

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

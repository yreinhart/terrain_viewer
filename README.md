# Terrain Viewer

Онлайн-сервис для анализа участка по XLSX-файлу с координатами `X`, `Y`, `H`.

## Возможности

- загрузка XLSX с точками высот;
- демо-участки без загрузки файла;
- карта высот и горизонтали;
- интерактивная и статическая 3D-модель;
- расчет объема подсыпки для выбранной зоны;
- учет коэффициента уплотнения грунта;
- расчет направления поверхностного стока D8;
- анализ подтопления по уровню воды;
- сохранение проекта в JSON и повторное открытие проекта.

## Формат XLSX

Файл должен содержать колонки:

| X | Y | H |
|---|---|---|
| 0 | 0 | -1.01 |
| 1 | 0 | -1.25 |

Где:

- `X` - координата по X, м;
- `Y` - координата по Y, м;
- `H` - отметка высоты, м относительно нуля проекта.

## Локальный запуск

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```
